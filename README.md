# Muthuraj P | Portfolio

A modern and professional personal portfolio website built with Next.js, designed to showcase my background, technical skills, experience, and featured projects.

## Overview

This portfolio highlights:
- My education and professional experience
- Core technical skills
- Selected projects and achievements
- Contact information for collaboration and opportunities

## Features

- Clean and responsive user interface
- Smooth animations and modern visual design
- Sections for About, Skills, Experience, and Projects
- Easy navigation for both desktop and mobile users

## Tech Stack

- [Next.js](https://nextjs.org/)
- [React](https://react.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Framer Motion](https://www.framer.com/motion/)
- [Lucide Icons](https://lucide.dev/)

## Getting Started

### Prerequisites

- Node.js
- pnpm

### Installation

```bash
pnpm install
```

### Run locally

```bash
pnpm dev
```

Then open http://localhost:3000 to view the portfolio in your browser.

## Available Scripts

```bash
pnpm dev      # Start the development server
pnpm build    # Build the project for production
pnpm start    # Run the production server
pnpm lint     # Run project lint checks
```

## Project Structure

- `src/pages` - Main pages and layout
- `src/styles` - Global styling and theme styles
- `public` - Static assets

## Deployment

This project can be deployed easily on platforms such as Vercel or Netlify.
